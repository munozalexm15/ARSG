# CONTROLS:
- WASD: Movement
- Left click: Shoot
- Right click: Aim
- R: Reload
- V: Melee (WIP)
- Z: Flashlight
- F: Interact / Grab ammo / Swap weapon
- Shift: Sprint
- Control: Crounch
- Shift + Crouch: Slide
- Spacebar: Jump
- Q / E: Leaning
